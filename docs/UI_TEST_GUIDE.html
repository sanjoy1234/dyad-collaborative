üîß Troubleshooting

Problem: Can't find where to type prompts

Solution: Make sure you've clicked on a project card and are on the /editor/{id} page, not the dashboard

Problem: Right panel is missing

Solution: Look for a collapse/expand button (arrow icon). The panel might be collapsed.

Problem: "Configure your model first" error

Solution: Click "Configure Model" and set up your API key

Problem: No response after sending prompt

Solution: Check API key is valid, has credits, and internet connection is working

Problem: Editor page shows blank screen

Solution:

  1. Open browser console (F12)
  2. Check for JavaScript errors
  3. Refresh page (Ctrl+R / Cmd+R)
  4. Check Docker logs: docker logs dyad-collaborative-app-1
